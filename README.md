## 🔥 O Problema Real

> *"Você sabe quem votar na próxima eleição? Você sabe qual canditado é mais a tua cara?"*

Com centenas de deputados federais, cada um com histórico, votos e posicionamentos distintos — **ninguém tem tempo de pesquisar todos**. O resultado? Votos por nome, por foto, por família política — raramente por afinidade real.

**O Match Eleitoral muda isso.**

## 🧠 Algoritmo de Afinidade

Cada alternativa das perguntas já está associada a determinados deputados. O sistema usa isso para calcular o alinhamento:

| Etapa | O que acontece |
|:-----:|----------------|
| **①** | Cada alternativa das perguntas está vinculada a deputados específicos |
| **②** | As respostas do usuário ativam os vínculos correspondentes |
| **③** | Os pontos se acumulam para cada um dos 20 deputados cadastrados |
| **④** | O ranking final ordena do maior para o menor score |

> 💡 **Simples de entender, poderoso na prática** — transparência total no resultado.
